# Contributing

Thank you for contributing to GPT4All.

## Contribution Guidelines

- Fork the repository
- Create a new branch
- Make your changes
- Commit your updates
- Submit a pull request

Please ensure your code is clean, documented, and properly tested before submitting.

---

# Code of Conduct

We are committed to maintaining a respectful and welcoming community.

Please:
- Be respectful
- Use professional language
- Accept constructive feedback
- Support other contributors

Unacceptable behavior includes:
- Harassment
- Offensive comments
- Personal attacks
- Sharing private information

Project maintainers reserve the right to remove inappropriate content or restrict access if necessary.
