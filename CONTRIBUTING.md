# Contributing

Please discuss any changes via issue or email before making a pull request.

All contributors must follow the Code of Conduct.

---

## Pull Requests

- Remove unnecessary build/install files before submitting  
- Clearly describe your changes  
- Add documentation for new or updated code  
- Include tests or demos for major changes if possible  
- All PRs require maintainer review before merging  

---

## Code of Conduct

Be respectful and professional at all times.

Unacceptable behavior includes harassment, discrimination, or any harmful conduct.
