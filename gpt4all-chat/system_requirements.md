# System Requirements

Below are the recommended and minimum system requirements for GPT4All.

---

## 🚀 Recommended System Requirements

| Component | PC (Windows / Linux) | Apple |
|----------|----------------------|-------|
| CPU | Ryzen 5 3600 or Intel i7-10700 (or better) | M2 Pro |
| RAM | 16 GB | 16 GB |
| GPU | NVIDIA GTX 1080 Ti / RTX 2080 or better (8GB+ VRAM) | Apple M2 Pro (integrated GPU) |
| OS | Windows 10+ or Ubuntu 24.04 LTS | macOS Sonoma 14.5+ |

---

## ⚙️ Minimum System Requirements

| Component | PC (Windows / Linux) | Apple |
|----------|----------------------|-------|
| CPU | Intel i3-2100 / Pentium 7505 / Celeron 6305 / AMD FX-4100 | Apple M1 |
| RAM | 16 GB (8 GB for small 3B models) | 16 GB |
| GPU | Direct3D 11/12 or OpenGL 2.1 capable GPU | Apple M1 (integrated GPU) |
| OS | Windows 10 / Ubuntu 22.04 LTS or compatible Linux | macOS Monterey 12.6 |

---

## ⚠️ Note

- Windows and Linux ARM systems are **not currently supported**.
